# ダーツ自宅練習用スクリプト

rubyの実践勉強とダーツのモチベーション維持のために作ってみました

自宅練習時のスコア管理用

## CLIから得点をスペース区切りで入力

- 例

```
# 19トリプル,20ダブル,18シングル
19t 20d 18
# ブル ダブルブル 1シングル
50 50d 1
```

## データの保存
- 結果をjsonへ保存

# countup.json

```
{
  2016-03-01 => [ 500, 400, 450... ],
  2016-03-02 => [ 500, 400, 450... ]
}
```

# award.json

```
{
  2016-03-01 => { ROWTON => 3, HIGHTON => 2, ...},
  2016-03-02 => { ROWTON => 3, HIGHTON => 2, ...}
}
```
